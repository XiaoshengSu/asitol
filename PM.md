## 一、核心架构

### 1.1 数据层
| 功能点 | 技术要点 | 优先级 |
|--------|---------|--------|
| 多格式树解析 | Newick/Nexus/PhyloXML/NHX 解析器，支持元数据标签 | P0 |
| 节点索引系统 | 唯一ID生成、父子关系映射、层级缓存 | P0 |
| 注释数据绑定 | 节点ID匹配校验、多源数据关联、动态更新 | P0 |
| 序列数据接口 | 对接FASTA/GenBank，支持特征提取 | P1 |

### 1.2 渲染引擎
| 功能点 | 技术要点 | 优先级 |
|--------|---------|--------|

| 画布系统 | SVG（矢量）+ Canvas（大数据）双模式 | P0 |
| 层级合成 | 树结构层 + 注释图层独立渲染、混合叠加 | P0 |

## 二、可视化功能模块

### 2.1 基础树形展示
```
- 分支样式：实线/虚线、粗细映射（支持置信度/进化距离）
- 节点标记：圆点/方块/三角形、大小/颜色变量映射
- 标签系统：叶节点标签、内部节点标签、标签碰撞检测与自动避让
- 比例尺：进化距离标尺、时间标尺（分子钟校准）
```

### 2.2 注释图层系统（核心差异化功能）
| 图层类型 | 数据映射 | 渲染方式 |
|---------|---------|---------|
| COLORSTRIP | 分类变量 → 色块 | 环形/矩形色带 |
| HEATMAP | 数值矩阵 → 色阶 | 网格单元格 |
| BARCHART | 数值 → 柱高 | 极坐标/直角坐标柱形 |
| PIECHART | 比例 → 扇形角度 | 堆叠扇环 |
| BINARY | 0/1 →  presence/absence | 实心/空心标记 |
| STRIP | 区间位置 → 色段 | 线性序列条带 |
| ALIGNMENT | 序列 → 碱基/残基颜色 | 序列比对可视化 |
| CONNECTIONS | 节点对 → 连线 | 贝塞尔曲线/直线 |
| POPUP | 富文本/图片/链接 | 交互浮层 |

### 2.3 交互功能
- **导航**：平移、缩放（鼠标滚轮/手势）、聚焦到节点、重置视口
- **选择**：单选/框选/多选节点、反向选择、选择集保存
- **折叠**：按层级折叠、按分支长度过滤、按注释值筛选
- **搜索**：节点ID模糊匹配、正则搜索、注释内容检索
- **联动**：圈选热图同步高亮、多树联动比对

## 三、数据导入/导出

### 3.1 输入接口
| 格式 | 解析要求 |
|------|---------|
| Newick | 标准语法 + 扩展标签（NHX: 支持进化事件标注）|
| Nexus | Blocks解析（TAXA/TREES/CHARACTERS）|
| PhyloXML | 完整Schema支持，保留元数据 |
| 注释文件 | TSV/CSV自动识别，列映射配置 |

### 3.2 输出能力
- **矢量图**：SVG（分层保留）、PDF（嵌入字体）
- **位图**：PNG/JPEG（分辨率DPI设置）、WebP
- **数据导出**：修剪后的子树（Newick）、筛选后的注释表、节点坐标数据
- **会话保存**：完整状态序列化（树+注释+视图参数）

## 四、编辑与计算模块

### 4.1 树操作
```
- 重新定根：指定外群、中点定根
- 分支操作：交换子节点顺序、删除/修剪分支、合并树（共识树构建）
- 长度校准： ultrametric转换（时间树）、等距归一化
- 拓扑校验：检测多叉节点、环状结构、零长分支
```

### 4.2 注释计算
- **数值聚合**：按内部节点汇总子树统计量（sum/mean/max）
- **颜色映射**：连续数值分段（等宽/等频/聚类）、离散变量自动配色
- **缺失值处理**：插值、隐藏、特殊标记

## 六、技术实现建议

### 6.1 前端技术栈
| 模块 | 推荐方案 |
|------|---------|
| 渲染引擎 | D3.js（灵活性）或 PixiJS（性能）+ 自定义Shader |
| 布局计算 | Rust/WASM 加速（大规模树）|
| 状态管理 | Zustand/Redux（复杂交互状态）|
| UI组件 | React/Vue + 自研可视化控件库 |

### 6.2 后端架构
- 树解析：BioPython/BioPerl 或自研解析器（Rust/Go）
- 注释处理：Pandas/Arrow 内存计算
- 缓存：Redis 存储预计算布局
- 存储：对象存储（原始文件）+ 图数据库（Neo4j，复杂查询）
